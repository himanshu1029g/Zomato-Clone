# Simple Zomato Clone 🍽️

**A small, personal Zomato-like UI built as a learning project over ~1–2 years.** It's a simple, front-end demo — good enough for showcasing layout and styling work.

---

## 🔧 Tech & Files
- **Built with:** HTML & CSS (static files).
- **Main files:** `index.html`, `just.html`, `style.css` and the `image/` folder containing assets like `miantop.avif`.

## ✨ Features
- Clean, responsive layout for restaurant listings
- Image-based cards and headers
- Simple navigation and demo pages

## ▶️ How to run
- Open `index.html` directly in your browser.
- Or run a quick local server (recommended for testing assets):


  - Or use VS Code Live Server extension and open the workspace.
